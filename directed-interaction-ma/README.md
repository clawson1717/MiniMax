# Directed Interaction Multi-Agent (DIMA)

A multi-agent coordination framework that combines Dynamic Interaction Graphs (DIG) with Logical Transduction Algebra (LTA) to create verifiable and explainable collaboration paths.

## 🚀 Concept
DIMA treats each agent interaction as a **typed "transducible function,"** allowing for structured data flow while maintaining the flexibility of dynamic graph-based decision paths. It ensures that emergent collaboration remains causally valid and type-safe from start to finish.

## 🧬 Papers Combined
1. **DIG to Heal** (Yang et al.) — Dynamic Interaction Graphs for explaining emergent collaboration and real-time error correction.
2. **Agentics 2.0** (Lee et al.) — Logical Transduction Algebra for formalizing agent calls as typed, explainable functions.
3. **LOGIGEN** (Zeng et al.) — Triple-agent orchestration (Architect, Set Designer, Explorer) for synthesizing complex, verifiable tasks.

## ✨ Novel Contribution
The key innovation is the **"Typed Interaction Path."** While DIG explains *how* agents collaborated, DIMA uses LTA to enforce *what* they can communicate at each node in the graph. This self-healing architecture identifies failing agent nodes and bypasses them using type-valid alternative paths.
