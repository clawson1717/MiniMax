# Proof-Narrative-Memory (PNM)

An agentic framework for **Automated Theorem Proving** that uses **Persuasive Narratives** to prioritize search space within an **Iterative Memory Caching** architecture.

## Overview
PNM combines three cutting-edge techniques from recent ArXiv research:
1.  **Iterative Proof Refinement:** A minimal agentic baseline for theorem proving.
2.  **Narrative Persuasion:** A heuristic layer using narrative features to guide proof trajectories.
3.  **Memory Caching:** A growing memory architecture for long-horizon mathematical reasoning.
