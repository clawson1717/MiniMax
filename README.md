# Resilient Adaptive Agent

Web agent with CATTS compute allocation, attack resilience, and checklist verification.

## Overview

This project combines three ArXiv techniques:
- **CATTS**: Dynamic compute allocation
- **Multi-Turn Attack resilience**: Defense against adversarial interactions
- **CM2**: Checklist-based rewards for verification
