# PsycheRAG — Need-State Driven Retrieval-Augmented Generation

> Retrieval should serve needs, not just match text.

## Overview

PsycheRAG reimagines RAG as a **psychologically-grounded memory system** where retrieval is driven by fundamental need states (survival, coherence, growth, social) rather than pure semantic similarity. Need urgency modulates retrieval depth, rule filters enforce action feasibility, and denoised trajectories provide high-quality candidates.

## Core Innovation

Combines three recent papers:
- **Computational Concept of the Psyche**: Formalizes AGI as operating in a state space of needs
- **RPMS**: Rule-augmented memory synergy for action feasibility filtering
- **OpenSeeker**: Fact-grounded synthesis and denoised trajectories
